# Watch Store Fullstack Project
Dự án quản lý cửa hàng đồng hồ sử dụng Flutter (Frontend) và Go (Backend).

## Công nghệ sử dụng
- **Backend:** Go (Golang), PostgreSQL
- **Frontend:** Flutter (Dart)
- **Database:** PostgreSQL với UUID & mã hóa pgcrypto

## Cấu trúc dự án
- `/backend`: Go source code (Branch: backend-dev)
- `/frontend`: Flutter source code (Branch: frontend-dev)
